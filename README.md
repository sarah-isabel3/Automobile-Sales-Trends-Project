# Automobile Sales Trends Project
This project conducts an exploratory data analysis (EDA) of global miniature automobile sales to uncover trends, understand customer purchasing behavior, and support data-driven marketing strategies. The analysis focuses on sales performance, product line impact, deal size patterns, and customer segmentation. Insights from the study help optimize stocking decisions and improve future business planning.

# Tools Used
Tableau – Dashboards & visualizations
Python / Jupyter Notebook – Data cleaning
